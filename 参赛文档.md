# Zil Desktop — 舒适自律，有温度的 AI

自律不是咬牙切齿地坚持，而是温柔地陪自己变好。

Zil Desktop 是一款主打「舒适」的桌面自律应用，把习惯打卡、日程安排、番茄专注、运动预设和 AI 洞察融进一个安静的窗口里。没有催命弹窗，没有社交压力，只有你和每天进步一点点的生活。

**它最不一样的地方：双层智能 + 一只陪你长大线条小猫——本地神经网络零依赖离线可用，云端大模型做个性化推荐，全程有温度的陪伴感。**

## 灵魂角色：线条小猫伙伴 Zili

侧边栏住着一只手绘线条风的小猫 Zili，它不是装饰，而是会随你状态变化的小伙伴：

- **四种表情随今日完成率切换**：打盹（0%）→ 陪着你（<50%）→ 开心（≥50%）→ 欢呼跳跃（100%）
- **打卡反馈联动**：你每完成一次打卡，Zili 会蹦跳一下并弹出气泡（"好棒！""继续加油~""喵~"），陪伴感拉满
- **全部 SVG 手绘**：圆头、三角耳、胡须、尾巴，纯 path + stroke 绘制，零图片依赖，呼吸动画让它一直"活着"

## 习惯成长树：可视化你的坚持

每个习惯左侧不再只是个图标，而是一棵会随连续打卡天数成长的线条小树：

- **5 个成长阶段**：种子（0 天）→ 嫩芽（1-2 天）→ 小树（3-6 天）→ 大树（7-29 天）→ 开花（30+ 天）
- 开花阶段会飘落彩色花瓣，呼应你长久的坚持
- 树冠随风轻摆，呼吸感与生命感十足
- 全部 SVG path 手绘，根据 streak 实时切换形态

## 本地神经网络层：纯 TypeScript 手写 MLP

我用纯 TypeScript 手写了完整的多层感知机（MLP），从矩阵乘法、ReLU、Softmax 到反向传播和交叉熵损失全部自己实现，含动量 SGD、mini-batch、L2 正则化、学习率衰减、早停机制和 Xavier 初始化。训练好的权重直接存到 localStorage，不需要云端、不需要 API Key、不需要网络。

为了让模型真正「懂你」，我把所有 AI 功能接入了一个**统一智能引擎**：

- **共享 10 维特征工程**：时段、星期、是否周末、连续天数、今日完成率、7 天平均完成率、近 3 天趋势、今日番茄数、历史平均专注时长、上次专注时长——一次构建，三个模型共享
- **三个独立 MLP 各司其职**：语录预测（10→12→6）、专注推荐（10→10→3）、习惯完成概率（14→12→2）
- **在线学习闭环**：每次打卡、每次专注、每次反馈都被记录为训练样本，达到 8 条就启动微调，权重实时落盘，越用越懂你
- **样本上限 200 条**的滑动窗口，保证模型始终跟踪最近的你而不是过去的你

4 个算法应用：

- **智能激励语**：根据时段、连续天数、完成率、一致性，自动选一句最懂你今天状态的话，并对每条反馈"有用/没用"实时强化模型
- **中断风险预测**：基于历史数据预测下周可能中断的概率，并给出个性化建议
- **专注时长推荐**：结合当前时段、今日番茄钟数、历史平均时长，推荐最合适的专注档位，记录你实际选择作为标签在线训练
- **习惯优先排序**：共享特征 + 习惯特定特征（连续天数、完成率、时段匹配度、创建天数）共同输入，预测今天每个习惯的完成概率，告诉你最该先做哪一件

## 云端大模型层：GLM-4-Flash 习惯推荐

接入智谱 GLM-4-Flash，根据你已有的习惯，AI 推荐互补的新习惯——不是泛泛的"多运动"，而是"晨间 10 分钟拉伸""午后冥想 5 分钟"这样具体可执行的建议，并附带推荐理由。一键采纳后自动加入习惯列表，图标、颜色、频率、提醒时间都已为你配好。

这是**可选功能**：用户在设置中填入自己的智谱 API Key 即可启用，不填完全不影响其他功能。API Key 帮助链接直通官方平台 `https://open.bigmodel.cn/apikey/platform`。

## 心情打卡：让 AI 更懂你的状态

每天首次打开应用时，会温柔地弹出心情打卡（5 档：很棒 / 不错 / 一般 / 不太好 / 糟糕），每档配一个手绘颜文字表情。心情数据本地持久化，并显示在首页统计胶囊中，作为 AI 理解你状态的额外信号。不想填可以"稍后再说"，绝不催促。

## 日程安排：月历看板

全新的日程板块，用月历网格 + 当日详情看板的形式组织每一天：

- 6 行 7 列月历，每个日期显示当天日程数量和分类色点
- 点击某天切换到当日详情看板，按时间顺序列出所有事件
- 5 种分类（工作 / 生活 / 学习 / 健康 / 其他）各自配色，一眼分清
- 空日子用手绘线条日历插画填充，告别冷冰冰的"暂无数据"

## 运动健身预设

精简后的预设模块聚焦于一件事——帮你动起来：

- 自定义锻炼类型与每日目标（分钟数）
- 一键记录每次运动，时间分配可视化
- 7 天柱状图统计本周运动趋势
- 环形进度环显示今日完成度

## 专注模式

输入当前要专注的任务，进入全屏沉浸状态，侧边栏自动隐藏，按 F11 / Esc 退出。专注完成时有柔和音效和视觉提示。整个界面保持简洁安静，不被打扰。

## 打卡反馈：粒子与音效

打卡成功的瞬间，按钮位置会飞出 8 个手绘线条粒子（星星 / 爱心 / 闪光），随机轨迹扩散消散，配合已有的打卡音效，给"完成"这个动作一次有仪式感的庆祝。同时触发侧边栏 Zili 蹦跳反馈。

## 手绘可爱风视觉系统

整个应用采用**专业手绘可爱风格**，把"陪伴感"贯彻到每一个像素：

### 设计语言

- **手绘字体体系**：标题用 Caveat（手写体）传达温度，正文用 Patrick Hand / Kalam（手写印刷体）保证可读，中文以 Noto Sans SC 兜底
- **暖白手工纸底色**（#FBF8F1）+ 全局 SVG 颗粒纹理叠层，营造真实纸面质感
- **柔和三色光晕**：薄荷青 / 奶油黄 / 珊瑚粉三色径向光晕，可爱不刺眼

### 不规则有机形态

所有 UI 组件摒弃标准几何形状，采用手绘不规则圆角（`radius-hand` 系列）：

- 卡片、按钮、输入框、导航项、徽章全部使用四角不对称的有机圆角
- 输入框采用手绘虚线描边，聚焦时转为实线，模拟手绘勾边过程
- 按钮采用实线描边 + 偏移硬投影（hatching 风格），按下时位移回弹

### SVG 手绘 filter 系统

在 index.html 内嵌一套 SVG 滤镜，仅在**装饰性元素**上施加位移抖动，营造手绘颤抖感：

- `handDrawn` / `handDrawnStrong`：feTurbulence + feDisplacementMap 实现的线条抖动
- `paperTexture`：纸面颗粒纹理
- `pencilSketch`：铅笔笔触边缘粗糙化
- `handShadow`：带抖动的偏移硬阴影

### 文字/数字清晰度保障

**关键工程决策**：SVG displacement filter 会让所有内容（包括文字数字）抖动失真。为确保"数值呈现准确且符合视觉规范"：

- 所有承载文字、数字的元素（卡片、按钮、统计胶囊、模态框、导航项、气泡）**一律不施加** displacement filter
- 手绘质感通过不规则圆角 + 偏移阴影 + 虚线描边 + 手写字体 + 纸面纹理等**非破坏性**手段实现
- filter 仅用于纯装饰 SVG（小猫 Zili、成长树、线条粒子、Logo 图标框、分隔线、导航活跃底色）
- 进度环 SVG 不加 filter，保证环形与百分比数字绝对精确

这样既保留了手绘可爱风格的温度，又保证了信息传达的准确性与专业度。

### 手绘装饰细节

- **小猫 Zili**：陪伴式线条角色
- **成长树**：每个习惯一棵，可视化坚持
- **粒子特效**：手绘星星 / 爱心 / 闪光
- **空状态插画**：笔记本 + 加号（习惯）、日历 + 星星（日程）、月亮 + Zzz（夜晚）等，全部 SVG path 绘制
- **心情颜文字**：每日心情用 unicode 颜文字表达
- **手绘下划线 / 分隔线**：标题用手绘波浪下划线强调，区块间用手绘波浪分隔线
- **模态框角标**：右上角手绘小星星点缀

## 现代 UI 设计

- **浮动玻璃拟态侧栏**：12px 外边距、大圆角、半透明背景 + backdrop-blur，分组导航（主要 / 工具），layoutId 弹簧动画的活跃指示器在导航项间流畅滑动
- **底部用户区**：头像（昵称首字母）+ 昵称 + 状态提示，一键跳转设置
- **移动端底部导航**：同样的玻璃拟态风格，所有功能在小屏上依然好用
- **居中弹窗**：所有模态框通过 createPortal 直接渲染到 body，不受祖先 transform 影响，永远在视口正中
- **个性化主题**：自定义主色、自定义背景图（含遮罩透明度、模糊度）、专注模式独立背景色、主页进度环中心自定义图片

## 数据完全由你掌控

- 习惯、日程、专注记录、心情、设置全部本地存储（localStorage）
- 支持一键导出 / 导入 JSON 备份
- 神经网络权重和行为样本也本地保存，可随时清除重置
- 清除数据时一并重置所有设置与预设
